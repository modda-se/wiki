# Nintendo Entertainment System

## Info
- **Manufacturer:** [Nintendo](https://en.wikipedia.org/wiki/Nintendo)
- **Release Date:** 
    - JP: July 15, 1983
    - NA: October 18, 1985
    - EU: September 1, 1986
- **CPU:** Ricoh 2A03 8-bit processor (MOS Technology 6502 core)
- **Sound:** RP2A03 (NTSC) and RP2A07 (PAL)
- **Video Signal:** 
    - Composite
    - RF Video
- **Region Locked:** Yes
