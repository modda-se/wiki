# Super Nintendo Entertainment System

## Info
- **Manufacturer:** [Nintendo](https://en.wikipedia.org/wiki/Nintendo)
- **Release Date:** 
    - JP: November 21, 1990
    - NA: August 23, 1991
    - EU: June 1992
- **CPU:** Ricoh 5A22 @ 3.58 MHz
- **Sound:** Nintendo S-SMP
- **Video Signal:** 
    - Composite
    - S-Video
    - RGB
    - RF
- **Region Locked:** Yes
